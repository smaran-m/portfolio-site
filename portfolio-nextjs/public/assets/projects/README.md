# Projects Assets

This directory contains images for the projects gallery.

## Directory Structure

Each project should have its own subdirectory with:
- `thumbnail.png` - The thumbnail image shown in the grid
- `image1.png`, `image2.png`, etc. - Full-size images for the modal viewer
